Projeto com foco em: React e Redux

Trivia Game

Desenvolvemos um jogo de perguntas e respostas baseado no jogo Trivia utilizando React e Redux. Atividades foram feitas em grupo, as funcionalidades de acordo com as demandas definidas em um quadro Kanban. Utilizamos um store Redux, criamos reducers, actions, dispatchers, para conectar os componentes React além de implementarmos actions assíncronas na aplicação React que faz uso de Redux.
